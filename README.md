# holbertonschool-chatgpt-introduction

## Files Overview

- **factorial.py** – Fixes an infinite loop in a factorial calculation script.
- **print_arguments.py** – Prints the command-line arguments passed to the script.
- **change_background.html** – Fixes a JavaScript error preventing a button from changing the background color.
- **mines.py** – Debugs a simple Minesweeper game implemented in Python.
- **factorial_recursive.py** – Adds documentation to a recursive factorial implementation.
- **checkbook.py** – Documents a simple checkbook management system.
- **tic.py** – Identifies and fixes logical errors in a Tic-Tac-Toe game.
